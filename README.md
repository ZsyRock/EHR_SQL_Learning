Lee, G., Hwang, H., Bae, S., Kwon, Y., Shin, W., Yang, S., ... & Choi, E. (2022). EHRSQL: A Practical Text-to-SQL Benchmark for Electronic Health Records. Advances in Neural Information Processing Systems, 35, 15589-15601.
